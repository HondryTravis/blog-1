# 正则表达式之（先行断言，后行断言）

## 先行断言

语法：（?=pattern）

案例：匹配字符串 x，但 x 后面必须是 y

```js
/x(?=y)/;
```

## 先行否定断言

语法：（?!pattern）

案例：匹配字符串 x，但 x 后面必须不是 y

```js
/x(?!y)/;
```

## 后行断言

语法：（?<=pattern）

案例：匹配字符串 x，但 x 前面必须是 y

```js
/(?<=y)x/;
```

## 后行否定断言

语法：（?<!pattern）

案例：匹配字符串 x，但 x 前面必须不是 y

```js
/(?<!y)x/;
```
