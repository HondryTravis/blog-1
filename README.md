记录工作中遇到的问题，平时看书的总结，面试题，以及一些个人思考。

访问地址：[yhlben.github.io](https://yhlben.github.io/)

## 使用

博客采用 vuepress 作为主框架，lint-md 作为语法检查工具。

1、开始写博客

```js
npm run dev
```

2、检查语法格式，并自动修复语法错误

```js
npm run lint:fix
```
